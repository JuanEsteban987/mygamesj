Acerca de esta Plataforma

Esta es una página creada para el uso, edición y cambio de archivos de imagen o audio. Nuestro objetivo es proporcionar una herramienta ligera pero potente basada íntegramente en tecnologías web. Aquí puedes transformar tus fotografías con filtros avanzados de post-procesamiento o manipular pistas de audio utilizando la API de Web Audio para efectos en tiempo real.
